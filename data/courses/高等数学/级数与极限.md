# 级数与极限

## 一、数列极限的定义

### $\varepsilon$-$N$ 定义

设 $\{a_n\}$ 为一数列，$A$ 为常数。若对任意给定的正数 $\varepsilon$，总存在正整数 $N$，使得当 $n > N$ 时有：

$$|a_n - A| < \varepsilon$$

则称数列 $\{a_n\}$ 收敛于 $A$，记为 $\lim_{n \to \infty} a_n = A$。

### 数列极限的基本性质

- **唯一性**：收敛数列的极限唯一。
- **有界性**：收敛数列必有界。
- **保号性**：若 $\lim a_n = A > 0$，则存在 $N$，当 $n > N$ 时 $a_n > 0$。
- **夹逼定理**：若 $a_n \leq b_n \leq c_n$ 且 $\lim a_n = \lim c_n = L$，则 $\lim b_n = L$。

## 二、级数的概念与收敛性

### 无穷级数

给定数列 $\{u_n\}$，称表达式 $\sum_{n=1}^{\infty} u_n = u_1 + u_2 + \cdots$ 为无穷级数。其部分和为：

$$S_n = \sum_{k=1}^{n} u_k$$

若 $\lim_{n \to \infty} S_n = S$ 存在，则称级数收敛于 $S$；否则称级数发散。

### 收敛的必要条件

若级数 $\sum u_n$ 收敛，则 $\lim_{n \to \infty} u_n = 0$（反之不然，如调和级数）。

## 三、常见级数判别法

### 1. 比较判别法

设 $0 \leq u_n \leq v_n$：
- 若 $\sum v_n$ 收敛，则 $\sum u_n$ 收敛。
- 若 $\sum u_n$ 发散，则 $\sum v_n$ 发散。

### 2. 比值判别法（达朗贝尔判别法）

设正项级数 $\sum u_n$，若 $\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = \rho$：
- $\rho < 1$ 时级数收敛；
- $\rho > 1$ 时级数发散；
- $\rho = 1$ 时判别法失效。

### 3. 根值判别法（柯西判别法）

设正项级数 $\sum u_n$，若 $\lim_{n \to \infty} \sqrt[n]{u_n} = \rho$：
- $\rho < 1$ 时级数收敛；
- $\rho > 1$ 时级数发散。

## 四、幂级数与泰勒展开

### 幂级数

形如 $\sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x + a_2 x^2 + \cdots$ 的级数称为幂级数。

**收敛半径**：$R = \lim_{n \to \infty} \left|\frac{a_n}{a_{n+1}}\right|$（若极限存在）。在 $|x| < R$ 内绝对收敛，在 $|x| > R$ 时发散。

### 常见泰勒展开

$$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots \quad (-\infty < x < +\infty)$$

$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots$$

$$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + \cdots \quad (|x| < 1)$$

## 五、例题

**例 1**：判断级数 $\sum_{n=1}^{\infty} \frac{1}{n^2}$ 的收敛性。

**解**：用比较判别法。因为 $\frac{1}{n^2} \leq \frac{1}{n(n-1)} = \frac{1}{n-1} - \frac{1}{n}$（$n \geq 2$），而右侧为收敛的裂项级数（部分和趋于 1），故原级数收敛。事实上 $\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$。

**例 2**：用比值判别法判断 $\sum_{n=1}^{\infty} \frac{n!}{n^n}$ 的收敛性。

**解**：$\frac{u_{n+1}}{u_n} = \frac{(n+1)! \cdot n^n}{(n+1)^{n+1} \cdot n!} = \frac{n^n}{(n+1)^n} = \left(\frac{n}{n+1}\right)^n = \frac{1}{(1 + 1/n)^n} \to \frac{1}{e} < 1$，故级数收敛。

**例 3**：求幂级数 $\sum_{n=1}^{\infty} \frac{x^n}{n}$ 的收敛半径。

**解**：$\left|\frac{a_n}{a_{n+1}}\right| = \frac{n+1}{n} \to 1$，故收敛半径 $R = 1$。在 $|x| < 1$ 内绝对收敛。当 $x = 1$ 时为调和级数（发散），$x = -1$ 时为交错调和级数（收敛）。
