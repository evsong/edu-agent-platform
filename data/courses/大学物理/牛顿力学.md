# 牛顿力学

## 一、牛顿三定律

### 牛顿第一定律（惯性定律）

任何物体在不受外力作用时，将保持静止或匀速直线运动状态不变。该定律揭示了惯性的概念：物体保持原有运动状态的性质。

### 牛顿第二定律

物体的加速度 $\vec{a}$ 与所受合外力 $\vec{F}$ 成正比，与质量 $m$ 成反比：

$$\vec{F} = m\vec{a}$$

这是经典力学的核心方程。当力随位置或时间变化时，需结合微积分求解运动轨迹。

### 牛顿第三定律

两个物体之间的作用力和反作用力大小相等、方向相反、作用在不同物体上：

$$\vec{F}_{12} = -\vec{F}_{21}$$

## 二、做功与能量

### 恒力做功

当力 $\vec{F}$ 恒定，物体位移为 $\vec{s}$ 时：

$$W = \vec{F} \cdot \vec{s} = Fs\cos\theta$$

### 变力做功（定积分的应用）

当力 $F(x)$ 随位置变化时，做功公式为：

$$W = \int_a^b F(x)\,dx$$

这是**定积分在物理中的直接应用**。力对位移的积分等于做的功，体现了数学与物理的深层联系。

**例**：弹簧力 $F = -kx$，将弹簧从自然长度拉伸 $x_0$ 做的功：

$$W = \int_0^{x_0} kx\,dx = \frac{1}{2}kx_0^2$$

### 功率

瞬时功率定义为功对时间的导数：

$$P = \frac{dW}{dt} = \vec{F} \cdot \vec{v}$$

## 三、动能定理

### 定理表述

合外力对物体做的功等于物体动能的变化：

$$W_{\text{合}} = \Delta E_k = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$

动能定理的证明需要用到牛顿第二定律和定积分：

$$W = \int_a^b F\,dx = \int_a^b ma\,dx = \int_{v_1}^{v_2} mv\,dv = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$

这再次体现了定积分在物理推导中的关键作用。

## 四、动量与动量守恒

### 动量

物体的动量定义为：$\vec{p} = m\vec{v}$。

牛顿第二定律的动量形式：

$$\vec{F} = \frac{d\vec{p}}{dt}$$

### 冲量

力在一段时间内的累积效果称为冲量：

$$\vec{I} = \int_{t_1}^{t_2} \vec{F}\,dt = \Delta\vec{p}$$

### 动量守恒定律

当系统所受合外力为零时，系统总动量守恒：

$$\sum \vec{p}_i = \text{常量}$$

典型应用包括碰撞问题和爆炸问题。

## 五、例题

**例 1**：质量为 2 kg 的物体在水平面上受 $F = 6t$（N）的变力作用从静止开始运动，忽略摩擦，求 3 s 末的速度。

**解**：由 $F = ma$，得 $a = 3t$ m/s$^2$。

$$v = \int_0^3 3t\,dt = \frac{3}{2}t^2\Big|_0^3 = \frac{27}{2} = 13.5 \text{ m/s}$$

**例 2**：质量 $m_1 = 3$ kg 的小球以 $v_1 = 4$ m/s 与静止的 $m_2 = 1$ kg 小球发生完全弹性碰撞，求碰后速度。

**解**：由动量守恒和动能守恒联立：

$$m_1 v_1 = m_1 v_1' + m_2 v_2'$$
$$\frac{1}{2}m_1 v_1^2 = \frac{1}{2}m_1 v_1'^2 + \frac{1}{2}m_2 v_2'^2$$

解得：$v_1' = \frac{m_1 - m_2}{m_1 + m_2}v_1 = 2$ m/s，$v_2' = \frac{2m_1}{m_1 + m_2}v_1 = 6$ m/s。
