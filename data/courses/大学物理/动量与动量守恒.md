# 动量与动量守恒

## 一、动量与冲量

### 动量

质量为 $m$、速度为 $\vec{v}$ 的质点的动量定义为：

$$\vec{p} = m\vec{v}$$

动量是矢量，其方向与速度方向相同。牛顿第二定律的动量形式为：

$$\vec{F} = \frac{d\vec{p}}{dt}$$

此形式比 $\vec{F} = m\vec{a}$ 更具普遍性，适用于变质量系统（如火箭推进）。

### 冲量

力 $\vec{F}$ 在时间 $[t_1, t_2]$ 上的累积效果称为冲量：

$$\vec{I} = \int_{t_1}^{t_2} \vec{F}\,dt$$

当力为恒力时简化为 $\vec{I} = \vec{F}\Delta t$。变力冲量的计算本质上是对力函数关于时间的**定积分**，这是微积分在力学中的又一重要应用。

## 二、动量定理

### 质点动量定理

合外力的冲量等于质点动量的变化：

$$\vec{I} = \int_{t_1}^{t_2} \vec{F}\,dt = \vec{p}_2 - \vec{p}_1 = \Delta\vec{p}$$

### 证明

由 $\vec{F} = \frac{d\vec{p}}{dt}$，两边对时间从 $t_1$ 到 $t_2$ 积分：

$$\int_{t_1}^{t_2} \vec{F}\,dt = \int_{t_1}^{t_2} d\vec{p} = \vec{p}_2 - \vec{p}_1$$

证毕。此处运用了**牛顿-莱布尼茨公式**，将微分方程转化为积分关系。

### 质点系动量定理

对质点系，内力冲量相互抵消（牛顿第三定律），故：

$$\sum \vec{I}_{\text{外}} = \Delta\vec{p}_{\text{系}}$$

## 三、动量守恒定律

当系统所受**合外力为零**时：

$$\sum_i \vec{p}_i = \text{常量} \quad \Rightarrow \quad \sum_i m_i \vec{v}_i = \sum_i m_i \vec{v}_i'$$

动量守恒定律的适用条件：(1) 系统合外力为零；(2) 某方向合外力为零时，该方向动量守恒；(3) 内力远大于外力时近似成立（如碰撞、爆炸）。

## 四、碰撞问题

### 完全弹性碰撞

动量和动能均守恒。设质量 $m_1$ 以速度 $v_0$ 碰撞静止的 $m_2$：

$$v_1' = \frac{m_1 - m_2}{m_1 + m_2}v_0, \quad v_2' = \frac{2m_1}{m_1 + m_2}v_0$$

特例：等质量碰撞（$m_1 = m_2$）时，$v_1' = 0$，$v_2' = v_0$，速度完全交换。

### 完全非弹性碰撞

两物体碰后粘合在一起运动（动能损失最大）：

$$v' = \frac{m_1 v_0}{m_1 + m_2}$$

动能损失率：$\frac{\Delta E_k}{E_{k0}} = \frac{m_2}{m_1 + m_2}$。

## 五、质心运动定理

### 质心

质点系的质心位置：

$$\vec{r}_c = \frac{\sum_i m_i \vec{r}_i}{\sum_i m_i}$$

### 质心运动定理

$$\vec{F}_{\text{外}} = M\vec{a}_c$$

其中 $M = \sum m_i$ 为系统总质量，$\vec{a}_c$ 为质心加速度。系统质心的运动如同一个集中了全部质量的质点在合外力作用下的运动。

## 六、例题与解答

**例 1**：质量 $m = 0.1$ kg 的球以 $v_1 = 5$ m/s 水平撞墙后以 $v_2 = 3$ m/s 反弹，接触时间 $\Delta t = 0.01$ s，求墙对球的平均力。

**解**：取初速度方向为正，$\Delta p = m(-v_2) - mv_1 = 0.1 \times (-3-5) = -0.8$ kg$\cdot$m/s。

$$\bar{F} = \frac{\Delta p}{\Delta t} = \frac{-0.8}{0.01} = -80 \text{ N}$$

负号表示力的方向与初速度相反，即指向球，大小为 80 N。

**例 2**：质量 $m_1 = 4$ kg 的物体以 $v_0 = 6$ m/s 与静止的 $m_2 = 2$ kg 物体发生完全非弹性碰撞，求碰后速度和动能损失。

**解**：由动量守恒：$v' = \frac{m_1 v_0}{m_1 + m_2} = \frac{4 \times 6}{6} = 4$ m/s。

动能损失：$\Delta E_k = \frac{1}{2}m_1 v_0^2 - \frac{1}{2}(m_1+m_2)v'^2 = 72 - 48 = 24$ J。

**例 3**：变力 $F(t) = 6t$（N）作用于质量 $m = 3$ kg 的静止物体上，利用冲量求 $t = 2$ s 时的速度。

**解**：冲量 $I = \int_0^2 6t\,dt = 3t^2\Big|_0^2 = 12$ N$\cdot$s。

由动量定理 $I = mv - 0$，得 $v = \frac{I}{m} = \frac{12}{3} = 4$ m/s。

## 七、习题

1. 质量 60 kg 的人以 2 m/s 跳上静止的 40 kg 小车，求人车共同速度。
2. 两球 $m_1 = 2$ kg（$v_1 = 3$ m/s）和 $m_2 = 1$ kg（$v_2 = -6$ m/s）发生完全弹性正碰，求碰后速度。
3. 变力 $F(t) = 4e^{-t}$（N）作用于 $m = 2$ kg 的静止质点，求 $t = 0$ 到 $t = \ln 2$ 期间的冲量和末速度。
