# EduAgent — 可嵌入式跨课程 AI Agent 通用架构平台

## 项目概要

当前高校教育智能化面临严重的碎片化问题：各课程独立开发AI辅助系统，导致重复建设、数据孤岛，教师与学生被迫在多个工具间切换，跨课程知识关联无从实现。

EduAgent 是一套面向高校教学场景的可嵌入式跨课程 AI Agent 通用架构平台。平台采用"统一框架 + 知识中间件 + 平台适配层"的三层架构，支持快速构建覆盖多课程的智能教学 Agent，并可无缝嵌入超星、钉钉等主流教学平台。

核心功能包括：基于 LangGraph Director Graph 的多 Agent 智能编排，由 Director 统一调度答疑、批改、辅导、分析、配置五个专业 Agent；位置级精细化作业批注引擎，通过 GPT-5.4 JSON Mode 结构化输出实现段落、字符级精准定位，批注准确率≥95%；基于贝叶斯知识追踪（BKT）算法的自适应练习推荐，构建"测-评-练"闭环；以及基于 Neo4j 知识图谱的跨课程知识迁移能力。

技术栈采用 Python FastAPI + LangGraph + Next.js + CopilotKit + PostgreSQL + Neo4j + Milvus + Redis，全容器化 Docker Compose 部署。系统支持500并发用户，知识点答疑正确率≥95%，兼容超星（LTI 1.3）与钉钉（开放平台SDK）两大教学平台。

本项目复用清华 OpenMAIC 编排模式、UC Berkeley OATutor BKT 算法等优质开源成果，在此基础上创新实现跨课程智能教学的完整闭环。
